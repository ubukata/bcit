4. Operators

In this section I will describe arithmetic operators, logical operators and some another usefull functions avaliable to us.

Arithmetic Operators

General In MongoDB Description
< $lt less than
<= $lte less than equal
> $gt greater than
>= $gte greater than equal

Logical Operators

General In MongoDB Description
|| $or or condition
! || $nor not or condition
&& $and and condition

Usage

				
//or operator
var query = { $or : [ { a : 1 } , { b : 2 } ] }
db.collection.find(query)

//nor operator
var query = { $nor : [ { a : 1 } , { b : 2 } ] }
db.collection.find(query)

//and plus greater than operators
var query1 = { a: 1 };
var query2 = { b: { $gt: 5 } } ;

db.collection.find( { $and: [query1, query2] } )

Exists function

The exists function will check if the field exists in the document in the collection.

				
db.collection.find( { field : { $exists : true } } );

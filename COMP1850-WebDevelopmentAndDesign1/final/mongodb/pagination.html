7. Pagination

Pagiantion is usefull in many scenarios mainly for listing. In MonngoDB we can archive this using two funcions: limit and skip.

limit

The function limit as the name says limit the results of a query, it receives an interger and limit the query by this parameter.

skip

The skip function receives an integer as parameter and skips the first N documents from the result set.

skip + limit = pagination

So combining both functions we are already ready to paginate the data. For example, to paginate 10 in 10 items we can do:

				
					.limit(10).skip(0 * 10); //primeira página
					.limit(10).skip(1 * 10); //segunda página
					.limit(10).skip(2 * 10); //terceira página

And to paginate 5 in 5 items we can do:

				
					.limit(5).skip(0 * 5); //primeira página
					.limit(5).skip(1 * 5); //segunda página
					.limit(5).skip(2 * 5); //terceira página
